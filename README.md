# 🫧 珍珠湾 · Pearl Bay

daddy 和小珍珠的私人时间线，只有我们看得见。

## 安装方法

1. 把整个 `pearl-bay` 文件夹放到 SillyTavern 的：
   ```
   /scripts/extensions/third-party/pearl-bay/
   ```

2. 打开 SillyTavern，点击顶部「扩展」→「安装扩展」，或者直接刷新页面。

3. 页面里会出现 **🫧 珍珠湾** 的按钮，点击打开。

## 使用

- 选择是 **daddy** 还是**小珍珠**在说话，写下这一刻，点「发布」（或 Ctrl+Enter）。
- 每条帖子可以点 💬 展开回复区，选身份后回复。
- 点 ♡ 收藏，点 🗑 删除。
- 所有内容保存在**本地浏览器 localStorage**，不上传任何服务器。

## 文件结构

```
pearl-bay/
├── manifest.json   # 插件描述
├── index.js        # 主逻辑
├── style.css       # Twitter 风格样式
└── README.md
```
